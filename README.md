# chip-8-emulator

Chip-8 emulator written in Rust.

## Setup

Clone the repository, then start the emulator using `cargo run`.

## Key map

```
1 2 3 C     1 2 3 4
4 5 6 D --> Q W E R
7 8 9 E     A S D F
A 0 B F     Z X C V
 chip8       local
```
